Webkit Flexbox Patterns

Useful for both prototypes and Chrome-only projects, the following examples demonstrate simple patterns of the more broadly supported, older 2009 flexbox specification — Chrome, Safari, and iOS Mobile Safari — on the left and the newer but poorly supported — Chrome only — 2012 specification on the right.

Basic flexbox example showing children with fixed widths.

HTML

<div class="example example-f">
    <div class="box box-1"></div>
    <div class="box box-2"></div>
    <div class="box box-3"></div>
</div>

2009

.example-f {
    display: -webkit-box;
}

.example-f .box {
    height: 100px;
    width: 100px;
}

2012

.example-f {
    display: -webkit-flex;
}

.example-f .box {
    height: 100px;
    width: 100px;
}
